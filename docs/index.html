Stats

Player stats, match details, and Elo history.

Get player stats and recent matches

Returns player metadata, aggregate record information, and a paginated list of matches for the selected filter. If the player is not found, the endpoint still returns HTTP 200 with player: null and empty stats.

path Parameters
username
required
string
Example: Feinberg

Minecraft username.

query Parameters
page
integer >= 1
Default: 1

One-based page number. Defaults to 1.

filter
string (MatchFilter)
Default: "competitive"
Enum: "competitive" "quick-play" "lobby"

Match queue filter. Defaults to competitive.

Responses

Draftout production server

https://draftoutmc.com/api/stats/{username}

Get a player's match detail

Returns player metadata and detailed match data for the requested match. If the match is not found for the player, the endpoint returns HTTP 200 with match: null.

path Parameters
username
required
string
Example: Feinberg

Minecraft username.

matchId
required
integer
Example: 20010

Numeric Draftout match ID.

Draftout production server

https://draftoutmc.com/api/stats/{username}/{matchId}

Ranks

Elo rank bands.

Get Elo rank bands

Returns the configured Draftout Elo rank bands in ascending Elo order. The lowest and highest bands are open-ended and use null for the missing bound.

Responses

Draftout production server

https://draftoutmc.com/api/ranks
